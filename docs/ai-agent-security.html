AI agent security

Scan AI-agent configs before permissions become incidents.

AgentFence helps developers catch risky agent access patterns early: broad filesystem permissions, destructive commands, secret-looking keys, unsafe runtime settings, and broad tool scopes.

Where AgentFence fits.

Local review

Run a scan before enabling new MCP servers or AI-agent tools in a project.

Pull requests

Fail CI when config changes introduce high or critical agent permissions.

Agency reports

Create lightweight risk reports for teams adopting AI coding workflows across multiple client repos.
