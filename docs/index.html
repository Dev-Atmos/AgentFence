Usage

Start with a local scan.

  --json agentfence-report.json \
  --sarif agentfence-report.sarif \
  --fail-on high

GitHub Action

Add AgentFence to pull requests.

- uses: Dev-Atmos/AgentFence@main
  with:
    path: .
    fail-on: high
    sarif: agentfence-report.sarif
