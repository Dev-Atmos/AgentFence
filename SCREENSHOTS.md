# Screenshot Checklist

Create these screenshots before wider launch:

1. Landing page:

```text
https://agentfence.pages.dev
```

2. Terminal command:

```bash
npx agentfence scan --path .
```

3. Fixture scan with reports:

```bash
npm run scan:fixtures
```

4. HTML report:

```text
agentfence-report.html
```

Recommended image names:

- `docs/assets/landing.png`
- `docs/assets/terminal-scan.png`
- `docs/assets/html-report.png`

Use these in README after the images are created.
